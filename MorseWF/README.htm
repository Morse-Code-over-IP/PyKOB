MorseWF Installation

You should allow at least half an hour to install MorseWF and associated system software. The good news is that this only needs to be done once per computer. Installing future updates to the program is a much simpler process.

The following instructions apply to all versions of Windows (XP, Vista, Windows 7, Windows 8 and 8.1, and Windows 10).

A) Download the MorseWF files
  1. Open the compressed file at sites.google.com/site/morsekob/morsekob40/downloads/MorseWF.zip.
  2. Copy the unzipped MorseWF folder to your hard drive.

B) Install Python and related software

  1. Download and install the latest version of Python 3 from python.org/downloads.
  2. Install pygame and pySerial (Python modules required by MorseWF) by double-clicking on the Install Modules batch file in the MorseWF folder.

C) Verify the installation

  1. Connect the key/sounder interface and the jack box to USB ports on the computer. Drivers for these devices may need to have been installed already.
  2. Open the MorseWF folder.
  3. Double-click on the SysCheck Run batch file. A command prompt window with a black background appears briefly while the program is running.
  4. The SysCheck program creates a new text file, SysCheck Data. Open this file by double-clicking on its icon. The contents should be something similar to the following:

    Python 3.3.2 (v3.3.2:d047928ae3f6, May 16 2013, 00:03:43) [MSC v.1600 32 bit (Intel)] on win32
    pygame 1.9.2a0
    pySerial 2.7
    MorseKOB 4.0-21
    Prolific USB-to-Serial Comm Port (COM3)
    Arduino Uno (COM4)
     
  5. Make a note of the comm port numbers for future reference.
D) Edit the config file

The MorseWF folder contains a configuration file named config.py. This file tells the MorseWF program which serial port to use for an external key and/or sounder, whether to use the speakers as a simulated sounder, and which port to use for the jack box.

Use Notepad to edit this file as required. (Right-click on the config file, and select Open with > Notepad.) A pound sign (#) at the beginning of a line tells the program to ignore that line. For example, the following config file would be used in the case where a key is connected to COM7, audio should be enabled to take the place of an actual sounder, and the jack box is connected to COM8.

# MorseWF configuration settings

# A line starting with a pound sign (#) is treated as a comment and is
# ignored by the system.

#PORT = None
PORT = 'COM7'

#AUDIO = True
AUDIO = False

#JACKPORT = None
JACKPORT = 'COM8'

FONTNAME = '' # default system font
#FONTNAME = 'arial'
#FONTNAME = 'timesnewroman'
FONTSIZE = 28

SCREENSIZE = None # default screen size
#SCREENSIZE = (1366, 768) # override automatic screen size
E) Edit the MorseMR-xx batch file

The MorseWF folder contains a number of batch files of the form MorseMR-xx. These batch files are used to launch the MorseWF program for office xx, and they contain the following command line parameters:

wire – default KOB wire no.
idText – office call, etc.
wpm – code speed

If you're using a jack box, you may want to edit the batch file to change the wire number to zero. This causes the program to run in standalone mode when the plug is removed from the jack box; otherwise, the program connects to the default wire when the plug is removed. The ID parameter needs to end with a pound sign (#) followed by a four-digit PIN code to give access to the Wells Fargo private wires. The code speed parameter determines the speed of the Morse sent by the computer when using the keyboard, and it should also be close to the fastest speed of any incoming Morse for the program to display decoded text reliably. Right-click on a batch file and select Edit to open it in Notepad if you wish to make any changes.

F) Create shortcuts

To make it easier to start up the program, you may wish to create desktop shortcut for the MorseWF-xx batch file. To do this, right-click on the batch file and select Send to > Desktop.

To launch the program automatically whenever the computer is booted, move the shortcut to the Windows Startup folder.
